1. Project Overview

YarnBop is a custom, interactive yarn and needle craft tracking app built with Flutter for Android.

Inspired by the interactive ePattern workflow of apps like Ribblr , YarnBop is designed to eliminate the "walled garden" restriction. It provides a dedicated digital crafting workspace where users can import external PDF patterns (from Etsy, Ravelry, or independent designers) and enjoy line-by-line row tracking, interactive overlays, and seamless progress saving without paywalls.


Primary Target User: Tina (designed as a personalized, daily-use crafting companion).


Project Nature: Personal use, educational learning project, and custom solution.

2. Core Experience & Philosophy
The core principle of YarnBop is frictionless crafting flow.

When a crafter is sitting with a crochet hook or knitting needles, the app should act as an intuitive, invisible assistant:


Universal Import: Any standard PDF pattern can be loaded instantly without manual copy-pasting or editor re-formatting.


Active Progress Tracking: Automatically save row counts, stitch counts, and page positions so the user can open the app and instantly resume where they left off.

Clean & Distraction-Free: Uncluttered UI designed for focused, cozy crafting sessions.

3. Initial Direction (Phase 1 Focus)
The immediate focus for YarnBop is delivering a rock-solid, local-first PDF pattern reader with interactive tracking tools:

Robust PDF document rendering optimized for mobile screens.

Persistent floating/overlay row and stitch counters paired with each pattern.

Movable line highlighter to keep focus on the active pattern instruction.

Project library view to manage active and completed patterns.

4. Long-Term Direction (Future Expansion)
Once the core PDF experience is perfected, YarnBop can evolve toward deeper interactive ePattern features:

Smart Text Parsing: Extracting text steps from imported PDFs into fully interactive, checkable list items.


Selective Size Highlighting: Allowing users to select their target garment size (e.g., Medium) and highlighting only the relevant stitch count numbers.


Crafting Toolbox: Built-in unit conversions (inches to cm) and needle/hook size conversion charts.


PDF Annotations: Adding custom sticky notes or handwritten markings directly onto pattern pages.


Yarn Stash Link: Tracking yarn colorways and gauge notes attached to specific PDF projects.
